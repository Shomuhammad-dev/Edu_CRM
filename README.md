# Alouddin Ta'lim Markazi CRM

This is the CRM system for Alouddin Ta'lim Markazi.

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app and backend together:
   `npm run dev`

If you prefer to run only the backend server, use:
   `npm run start`
